Описание программы загрузки видео и аудио с YouTube
Описание программы
Программа представляет собой графический загрузчик видео и аудио с YouTube, написанный на Python с использованием библиотеки yt-dlp и интерфейса Tkinter. Она позволяет скачивать одно видео или один аудиофайл в выбранном качестве, объединяя видео и звук в единый файл.
Основные шаги работы
1.	1. Ввод ссылки на видео.
2.	2. Проверка наличия ffmpeg.
3.	3. Выбор папки для сохранения.
4.	4. Настройка параметров загрузки (режим видео или аудио, качество).
5.	5. Запуск загрузки через yt-dlp.
6.	6. Отслеживание прогресса и обновление прогрессбара.
7.	7. Вывод сообщения об успешной загрузке или ошибке.
Используемые технологии
- Python
- Tkinter (GUI)
- yt-dlp (скачивание видео/аудио)
- ffmpeg (объединение видео и аудио)
Итог
Программа предоставляет удобный графический интерфейс для скачивания видео и аудио с YouTube, позволяя выбрать режим, качество и папку сохранения, а также отслеживать прогресс загрузки.
